# Канаты и зиплайны

🧵 Теперь вы можете создавать различные канаты — как для декора, так и для перемещения себя и мобов.

Канаты устанавливаются от точки A до точки B. Канаты можно соединять между собой — для этого нажимайте именно на якорные точки канатов.

## Где создать канаты

Сначала создайте специальную **станцию канатов** — на ней создаются все канаты.

![Станция канатов](../assets/images/rope-station.png)

При создании каната можно выбрать степень провисания — насколько сильно канат будет провисать.

## Декор для канатов

На канат можно повесить:

* фонари;
* аметисты;
* медные големы;
* надписи.

![Материалы для канатов](../assets/images/rope-materials.png)

## Зиплайн

🛝 Чтобы покатиться по канату под действием гравитации, создайте **Зиплайн Пад**. Установите его под канат на расстоянии до 3-х блоков. Встав на плиту, вы окажетесь на канате и покатитесь вниз.

![Зиплайн Пад](../assets/images/zipline-pad.png)

Плита ставится в начало пути зиплайна.

## Улучшение «Лифт»

⬆️ С этим улучшением вы сможете задать направление движения по канату — например, чтобы катиться вверх вопреки гравитации.

1. Создайте улучшение на верстаке.
2. Установите его на канат на каждую якорную точку в нужном направлении.

![Улучшение «Лифт»](../assets/images/rope-lift.png)
