# Быстрые счастливые гасты

💨 Ускорение ваших гастов с помощью зачарования Swift Fly.

## Уровни зачарования

| Уровень | Эффект |
|-------|-------|
| I | +40 % скорости |
| II | +60 % скорости |
| III | +80 % скорости |

## Крафт зачарований

**1‑й уровень:** зелье стремительности + обычная книга.

**3‑й уровень:** незеритовое улучшение + книга 2‑го уровня.
