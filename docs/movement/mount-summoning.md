# Призыв маунта

Уникальная механика призыва транспорта, вдохновлённая The Witcher.

## Как это работает

1. Получите **козий рог**.
2. Переименуйте маунта с помощью бирки.
3. Возьмите рог в руку.
4. Присядьте и посмотрите на питомца — рог зачаруется и получит кличку животного.
5. Для призыва подуйте в рог.

> **Внимание:** Будьте осторожны, чтобы не призвать питомца в блоки!

## Поддерживаемые маунты

* лошади;
* наутилусы.
