# Фермер

🌾 Уникальное зачарование для фермеров. Увеличивает шанс получить больше культур при сборе урожая.

Зачарование **нельзя получить обычным способом** — его можно только **выторговать у библиотекаря**.

## Уровни

Зачарование имеет 3 уровня:

| Уровень | Эффект |
|---|---|
| **I** | 25% шанс удвоения культур |
| **II** | 35% удвоения культур + 10% утроения культур |
| **III** | 50% удвоения культур + 20% утроения культур |
