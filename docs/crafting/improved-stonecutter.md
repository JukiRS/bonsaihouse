# Улучшенный камнерез

⚒ Расширенная версия ванильного камнереза с поддержкой новых материалов.

## Что можно обрабатывать

| Материал | Результат |
|--------|-----------|
| Обычные и обтёсанные брёвна | Разделка на доски |
| Доски | Декоративные элементы |
| Аметисты | Резные аметисты |
| Глубинный сланец | Плиты, ступени |
| Стекло | Витражи |
| Медь | Декоративные медные блоки |

## Преимущества

* экономия ресурсов;
* больше вариантов декора;
* совместимость с ванильными рецептами.
