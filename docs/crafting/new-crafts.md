# Новые крафты


🛠️ Расширенный список рецептов для улучшения игрового процесса.


## Кожа из гнилой плоти

Переплавьте гнилую плоть в печи для получения кожи.


## Чёрный краситель

Создаётся из каменного и древесного угля (1:1).


## Энергорельсы из меди


Используйте медь вместо золота для создания энергорельсов.

## Невидимые рамки


* **Невидимая рамка** — для декора.
* **Светящаяся рамка** — освещает пространство.
* **Обычная рамка** — стандартный вариант.

## Невидимый свет

Создайте невидимые блоки света (требуется жёлтое стекло) для освещения без факелов.

## Разжатие блоков


Теперь можно разжать:
* лёд;
* нарост;
* нитки.

Упрощает хранение и транспортировку.

## Кораллы из ростков

Скрафтите блоки кораллов из их ростков.
