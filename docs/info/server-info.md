# Информация о сервере

Главное, что нужно знать, чтобы зайти и начать играть на **Bonsai House**.

## Как подключиться

| Параметр | Значение |
|---|---|
| 🌳 Текущий сезон | **3** |
| 🐤 Версия | **Java 1.21.11** |
| 🌐 Основной IP | `mc.bonsaihouse.ru` |
| 🌐 Запасной IP №1 | `bonsai.vanilla.house` |
| 🌐 Запасной IP №2 | `bonsai.vanilla.cool` |

## Перезапуск сервера

🔄 Сервер уходит на перезапуск каждый день в **12:00** и **00:00** по МСК.

Перезапуск длится пару минут — просто подождите и заходите снова.

## Голосовой чат

🎙️ На сервере реализован голосовой чат через **PlasmoVoice**.
