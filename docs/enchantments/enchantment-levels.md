# Усиление чар

⚡ Расширенные уровни зачарований через наковальню.

## Доступные уровни

| Зачарование | Максимальный уровень | Эффект |
|-----------|------------------|-------|
| Защита | V | +12,5 % защиты |
| Эффективность | VI | +30 % скорости добычи |
| Прочность | V | +40 % долговечности |
| SwiftFlight | III | +80 % скорости гастов |

## Как усилить


1. Книга зачарования предыдущего уровня.
2. Незеритовое улучшение.
3. Объедините на наковальне.

> **Совет:** Используйте для создания топового снаряжения!
