# The Overworld

## Особенности генерации

🌍 Мир представляет собой остров размерами $6 \times 6$ тыс. блоков, окружённый бесконечным океаном.

### Ключевые особенности:

* кастовая генерация ландшафта и пещер;
* все ванильные биомы и структуры;
* дополнительные кастомные структуры;
* ограниченные ресурсы из‑за размера острова.

## Поиск крепости в Энд

🔎 Крепость **нельзя** найти с помощью ока края. Однако её можно обнаружить через исследование затерянных сооружений — копать под землёй не требуется.
