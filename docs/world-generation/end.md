# The End

## Особенности измерения

✨ Попасть в Энд можно через крепость в Overworld, которая была значительно изменена:

* структура расширена и усложнена;
* 23 новых биома вместо пустоты с островами;
* сложная система лута;
* механика «проклятых алтарей»;
* новые артефакты, оружие и броня.
