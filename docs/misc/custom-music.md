# Музыка на пластинках

🎵 Слушайте свои треки на сервере!

## Как записать

1. Найдите трек на SoundCloud.
2. Скопируйте ссылку.
3. Введите команду: `/disc burn [ссылка]`.
4. Пластинка появится в инвентаре.

## Воспроизведение

Поместите пластинку в проигрыватель.

Ссылка на SoundCloud: [soundcloud.com](https://soundcloud.com)
