# Тотем бессмертия

💀 Уникальный предмет для выживания в опасных ситуациях.

## Где найти

Выпадает с заклинателя с шансом 15 %.


## Использование

Активируется при получении смертельного урона — спасает жизнь один раз.

> **Внимание:** Шанс выпадения низкий — цените каждый тотем!
