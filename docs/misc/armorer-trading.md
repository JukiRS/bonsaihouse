# Торговля с бронником

💰 Бронник на последнем уровне (мастер) продаёт кузнечные шаблоны.

## Ассортимент

Включает все шаблоны, кроме:
* Око;
* Шпиль;
* Прилив;
* Тишина;
* Хранитель;
* Незеритовое улучшение;
* Рыло;
* Ребро.

> **Совет:** Планируйте развитие бронника для доступа к редким шаблонам!
