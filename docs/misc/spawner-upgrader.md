# Улучшитель спавнеров

�� Улучшайте параметры найденных спавнеров мобов.

## Предмет

«Улучшитель спавнеров» — специальный предмет для модификации.

## Эффекты улучшения

* скорость спавна +50 %;
* максимальное количество мобов за цикл: с 4 до 6.


> **Важно:** Используйте на спавнере ПКМ с улучшителем в руке.
