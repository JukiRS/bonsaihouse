# Подписывание предметов

📝 Персонализируйте своё снаряжение с помощью подписей.

## Как подписать

Используйте команду `/sign`.

## Типы подписей

* обычные (серые);
* цветные;
* редкие градиентные.

## Сброс подписи

Команда `/signreset` удалит подпись с предмета.

> **Совет:** Подписывайте ценные вещи, чтобы избежать путаницы!
