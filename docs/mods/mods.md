# Моды

🧩 Список модов для игры на **Bonsai House**. Версия игры — **Java 1.21.11**.

Скачайте нужный мод под свой загрузчик и положите `.jar` в папку `.minecraft/mods`. Для запуска нужен установленный **Fabric** или **NeoForge**.

ℹ️ На версии 1.21 классический **Forge** почти не поддерживается — его место занял **NeoForge**. Поэтому у части модов ниже доступны только сборки **Fabric** и **NeoForge**.

---

## Необходимые моды

Эти моды нужны для корректного отображения кастомных текстур и моделей сервера.

Не обязательны, но улучшают игру на сервере.
